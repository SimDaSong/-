# 지옥에서 온 Git - 수업소개

- Git = Version Control System(버전 관리 시스템)에 속하는 프로그램 중 하나
- Git의 역할
  1. 파일의 이름을 바꾸지 않고도 버전 관리를 할 수 있도록 도와줌
  2. 소스 코드를 백업할 수 있음
  3. 코드를 이전 상태로 쉽게 되돌려줌
  4. 협업이 가능케 함
- 버전 관리 시스템의 예
  - CVS : 예전에 많이 사용되었음
  - SVN : 예전에 많이 사용되었고 현재도 많이 사용되고 있음
  - GIT : 오늘날 가장 많이 사용됨
  - DropBox, Google Drive : 개발을 하지 않는 일반인들도 쉽게 사용 가능
